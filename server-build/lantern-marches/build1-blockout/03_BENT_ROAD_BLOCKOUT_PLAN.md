# The Bent Road Blockout Plan

## Purpose

The Bent Road is the player's first route out of safety. It should make the world feel old, fractured, and slightly wrong.

## Gameplay Role

- First directed path from Lantern's Rest
- First visual hint of RE:FRAGMENT damage
- Safe travel route with mild danger at edges
- Connects hub to Hollowglass Pool

## Recommended Shape

```text
Length: 250-350 blocks
Width: 5-9 blocks
Safe central path: 3-5 blocks
Side encounter pockets: 2-4 small areas
Sightline breaks: every 60-90 blocks
```

## Required Features

| Feature | Function |
| --- | --- |
| Bent stone road | Main navigation line |
| Uneven broken road pieces | Shows the world was damaged |
| Lantern posts | Wayfinding and starter safety |
| Abandoned cart / broken marker | Environmental storytelling |
| First anomaly shimmer | Foreshadow Hollowglass Pool |
| Road split that loops back visually | Avoid corridor feeling |

## Route Design Rule

The player should always know the main route, but should see 1-2 optional curiosities.

Bad version:

```text
Straight road, nothing else.
```

Better version:

```text
Main road curves and remains readable, while ruined side markers imply future exploration.
```

## WorldGuard Intent

The Bent Road gets a route region, but not a full safe zone.

Suggested behavior later:

- PvE allowed
- PvP disabled if starter world rules require it
- Low-level mobs only near edges
- Roads remain navigable
