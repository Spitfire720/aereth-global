# Build 01 Blockout Start Here

## Objective

Create the first playable physical slice of Aereth:

- **Lantern's Rest**: starter hub / registry / safe settlement
- **The Bent Road**: first traversal corridor and visual world hook
- **Hollowglass Pool**: first low-risk Fragment disturbance site

This is a blockout, not final art. The goal is to create playable space, not decorative perfection. Minecraft builders, naturally, will try to detail every lantern before the road exists. Do not let them.

## Success Criteria

The blockout is successful when:

- A player can spawn or enter Lantern's Rest safely.
- The player can visually understand where to go next.
- The Bent Road clearly leads away from the hub.
- Hollowglass Pool reads as an anomaly site.
- WorldGuard regions can be defined using captured coordinates.
- A basic 10-minute walk/playtest can be completed without confusion.

## Do Not Build Yet

Do not build these yet in Build 01:

- Full Lantern Marches region
- Complete villages
- Full quest interiors
- Final custom models
- Full mob camps
- Full dungeon entrances
- All roads to Valterra / Lyssara

Build 01 is just the skeleton. The beautiful corpse gets skin later.

## Canon Constraints

- The Lantern Marches are a starter subregion, not a continent.
- They sit between Valterra and Lyssara influence.
- Lantern's Rest is the main starter hub.
- The area should feel stable but surrounded by memory weirdness.
- The world beyond must feel larger and more dangerous.
- Fragments are not items.
- Intent and Discipline systems must not be represented as gear classes.
