- [ ] Confirm world name.

- [ ] Confirm WorldGuard is working.
- [ ] Confirm backup exists.
- [ ] Mark Lantern's Rest center coordinate.

## Lantern's Rest

- [ ] Central lantern landmark placed.
- [ ] Registry/ledger building blocked out.
- [ ] Archive post blocked out.

- [ ] Broken signboard placed.

## The Bent Road

- [ ] Main route is readable.
- [ ] Road curves or bends visibly.
- [ ] Lantern markers placed.
- [ ] 1-2 side curiosity pockets added.
