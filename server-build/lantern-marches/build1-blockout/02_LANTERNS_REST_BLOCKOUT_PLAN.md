# Lantern's Rest Blockout Plan

## Purpose

Lantern's Rest is the first safe hub. It teaches the player that Aereth is wounded, but not completely lost.

## Gameplay Role

- Spawn or arrival point
- Remnant registry point
- First Archivist contact
- First road marker toward the wider West
- Safe trading and orientation space

## Blockout Shape

Recommended rough footprint:

```text
Total hub: 90 x 90 blocks
Core plaza: 35 x 35 blocks
Registry building: 18 x 14 blocks
Archive post: 14 x 12 blocks
Road gate / lantern arch: 12 x 8 blocks
Outer palisade / low wall: partial, not full fortress
```

## Required Landmarks

| Landmark | Function | Notes |
| --- | --- | --- |
| Central Lantern | Orientation | Large enough to be seen from all hub entrances |
| Registry Ledger Hall | Character/Remnant registration | Later connects to BetonQuest + FragmentEngine |
| Small Archive Post | Lore intro | Archivist NPC location |
| Trade Stall Row | Recovery/frontier trade culture | Later can hold vendors |
| Roadward Gate | Exit toward The Bent Road | Strong visual pull |
| Broken Signboard | Shows world is larger | Points toward Valterra, Lyssara, Old Roads |

## Feel

- Safe, but not rich.
- Repaired, not pristine.
- Lanterns, old stone, wood beams, worn roads, patched roofs.
- Blue-teal memory glow should be rare and meaningful, not everywhere.

## Avoid

- Giant castle hub
- Overly clean medieval town
- Cyberpunk overload
- Massive verticality in starter area
- Too many NPC stalls before gameplay exists

## First Pass Build Palette

Use temporary materials first:

- Stone bricks / cracked stone bricks
- Spruce / dark oak
- Coarse dirt / path blocks
- Lanterns / soul lanterns sparingly
- Deepslate accents
- Blue stained glass or cyan glass for anomaly hints
- Moss and vines for age

Final palette is in `palettes/Build_01_Palette_First_Pass.md`.
