# Build 01 Palette - First Pass

This is a blockout palette. It is meant to be fast, readable, and easy to replace later.

## Settled West / Lantern's Rest

| Use | Blocks |
| --- | --- |
| Roads | dirt_path, coarse_dirt, gravel, cobblestone |
| Old stone | stone_bricks, cracked_stone_bricks, mossy_stone_bricks |
| Wood | spruce_planks, spruce_log, dark_oak_log, stripped_spruce_log |
| Roofs | deepslate_tiles, dark_oak_stairs, spruce_stairs |
| Light | lantern, soul_lantern sparingly |
| Age | moss_block, vines, cracked blocks |

## Bent Road

## Hollowglass Pool

| Use | Blocks |
| --- | --- |
| Pool | water, dark prismarine optional, black stained glass below water |
| Shards | blue_stained_glass, cyan_stained_glass, tinted_glass |
| Scar terrain | blackstone, polished_blackstone, deepslate, cracked_deepslate_tiles |
| Strange growth | dead_bush, mangrove_roots, hanging_roots |
| Glow hint | sea_lantern hidden, soul_lantern, end_rod sparingly |

## Rule

Use bright magic blocks sparingly. If everything glows, nothing feels magical. A lesson UI designers somehow refuse to learn.
