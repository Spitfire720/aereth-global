# Hollowglass Pool Blockout Plan

## Purpose

Hollowglass Pool is the first low-risk Fragment disturbance site. It introduces memory/reflection weirdness without making the player feel like they accidentally joined a PhD seminar in metaphysical suffering.

## Gameplay Role

- First anomaly investigation
- First Roadstray / minor memory-touched encounter area
- First place where Fragment lore feels physical
- First return objective back to Lantern's Rest

## Recommended Footprint

```text
Outer area: 70 x 70 blocks
Pool core: 18-25 block diameter
Inner anomaly ring: 30 x 30 blocks
Outer encounter ring: 55 x 55 blocks
```

## Required Features

| Feature | Function |
| --- | --- |
| Reflective pool | Visual focus |
| Hollowglass shards | Oraxen prop target later |
| Broken shrine/outpost remnant | Lore hook |
| Strange reflection line | Visual RE:FRAGMENT hint |
| Safe observation edge | Player can approach without immediate danger |
| Encounter pockets | Roadstray/memory mob spawn points later |

## Visual Direction

- Dark shallow water
- Blue/cyan glass shards
- Blackstone/deepslate fragments
- Sparse dead trees or bent roots
- Mist / fog if available through environment effects
- Not too large, not dungeon-sized

## Region Layers

| Region ID | Purpose |
| --- | --- |
| lm_hollowglass_pool_outer | General anomaly zone |
| lm_hollowglass_pool_inner | Core interaction / later quest target |
| lm_hollowglass_pool_spawn_edges | Optional mob spawn control area |

## Warning

Do not make Hollowglass Pool too epic. This is the first anomaly site, not the final boss arena where reality files for bankruptcy.
