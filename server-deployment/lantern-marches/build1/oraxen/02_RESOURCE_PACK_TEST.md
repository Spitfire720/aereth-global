# Oraxen Resource Pack Test - Build 01

## Goal

Confirm the Lantern Marches prop visuals appear correctly for players.

## Checks

1. Join with a clean client profile.
2. Accept/download the resource pack.
3. Confirm no missing purple/black texture squares.
4. Confirm each prop has the expected model/icon.
5. Confirm placed furniture appears to other players.
6. Confirm relogging preserves visuals.
7. Confirm pack send command works:

```text
/oraxen pack send <player>
```

## Known failure signs

- Item appears as default vanilla item.
- Placed furniture is invisible.
- Model appears huge or rotated incorrectly.
- Console errors mention missing model path or texture path.
- Pack downloads but client does not apply it.
