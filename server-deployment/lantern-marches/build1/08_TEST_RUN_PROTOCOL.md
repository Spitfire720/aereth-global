# Build 01 Test Run Protocol

## Test profile

Use a normal test player, not an operator account, for the actual flow.

## Test path

1. Spawn/arrive at Lantern's Rest.
2. Confirm hub protections.
3. Talk to Archivist Maerin.
4. Trigger registration dialogue.
5. Walk to The Bent Road.
6. Talk to Road Warden Tovan.
7. Follow road to first anomaly marker.
8. Spawn/fight one Roadstray.
9. Approach Hollowglass Pool.
10. Confirm Hollowglass Wisp behavior.
11. Return to Lantern's Rest.
12. Confirm journal/objective progression.

## Acceptable Build 01 quality

Build 01 can be visually rough.

Build 01 cannot be structurally confusing.

Players should understand:

- Where they are.
- Why the world is broken.
- Where to go first.
- What kind of game Aereth is trying to be.
