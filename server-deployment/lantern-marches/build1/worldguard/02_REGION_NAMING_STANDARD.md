# Lantern Marches WorldGuard Region Naming Standard

## Prefix

All Build 01 Lantern Marches regions use:

```text
aereth_lm_
```

Where:

- `aereth` = project/world namespace
- `lm` = Lantern Marches

## Naming rules

- Lowercase only.
- Use underscores, not spaces or hyphens.
- Keep names stable once configs or quests reference them.
- Do not use generic names like `spawn`, `road`, `town`, or `test`.
- Do not include version numbers in region IDs unless the region is temporary.

## Approved Build 01 names

aereth_lm_build1_outer
aereth_lm_lanterns_rest

aereth_lm_bent_road
