# Oraxen Starter Asset Notes - Lantern Marches Build 01

## Assets

- `lantern_registry_ledger`
- `bent_road_marker`
- `hollowglass_shard`
- `lantern_march_waystone`
- `sealed_ash_gate_marker`

## Boundary rule

These are visual/interaction assets only.

They must not become:

- Fragments
- Intent states
- Disciplines
- Titles
- race definitions
- permanent RPG identity

## First visual pass

Use placeholder generated models first. Replace with proper models later.

Suggested visual language:

- registry ledger: dark leather, teal glyph, gold corner plates
- bent road marker: cracked stone/wood sign, faint teal route mark
- hollowglass shard: transparent cyan-purple shard
- waystone: lantern-stone hybrid, frontier route symbol
- sealed ash gate marker: blackened post with red-orange ash seal
