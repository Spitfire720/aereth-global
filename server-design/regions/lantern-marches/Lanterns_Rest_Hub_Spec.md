# Lantern's Rest - Hub Spec

## Purpose

Lantern's Rest is the first hub players encounter in Aereth. It should teach the player what kind of world this is without making them feel trapped in a tutorial cage like a lab rat with a diamond sword.

## Hub identity

A rebuilt frontier settlement around an old road registry and lantern tower. It survives by maintaining routes between Valterra and Lyssara influence.

## Required hub features

| Feature | Gameplay use | Notes |
| --- | --- | --- |
| Arrival platform / road gate | Spawn point | Should face toward Lantern Tower |
| Lantern Tower | Visual anchor | Tall but not gigantic; warm light, teal anomaly cracks nearby |
| Registry Hall | Character/race/Remnant onboarding | Archivist NPCs, records, first dialogue |
| Market Row | Starter vendors | Food, low-tier tools, basic gear, repair hints |
| Roadwarden Post | Combat/tutorial quests | Sends player to Bent Road / Old Wick Post |
| Quiet Ledger Annex | Lore and Fragment hooks | Leads to Hollowglass Pool investigation |
| Waystone Plaza | Future travel unlock | Inactive or limited initially |
| Notice Board | Repeatable tasks | Later bounty/delivery hooks |
| Small shrine/crossing marker | Cultural ambiguity | No confirmed gods, only local belief |

## Mood

Lantern's Rest is safer than the world outside, but it should still show visible damage from RE:FRAGMENT:

- Roads bend slightly wrong.
- Some signs have overwritten names.
- A few windows reflect people who are not standing there.
- Old records contain missing entries.
- Lanterns flicker blue-teal near anomalies.

## Build palette

- Oak, spruce, stripped dark oak.
- Stone brick, cracked stone brick, tuff, andesite.
- Copper accents for registry fixtures.
- Iron chains and hanging lanterns.
- Amethyst/prismarine/cyan glass for controlled anomaly visuals.
- Gold accents only for official registry/Cartotheca/Archivist signs.

## Hub layout

Suggested layout:

```text
[North Gate / Bent Road]
        |
 Roadwarden Post ---- Market Row
        |              |
 Lantern Tower ---- Waystone Plaza
        |              |
 Registry Hall ---- Quiet Ledger Annex
        |
[Southwest Road / Lyssara Route]
```

## NPC groups

- Archivists / registry staff.
- Roadwardens.
- Traders and route carriers.
- Local settlers.
- One Concord-aligned healer/mediator.
- One suspicious traveler connected to The Severed, not exposed immediately.

## Hub rules

- No hostile mobs inside core.
- No player progression choices that permanently lock the player yet.
- No Discipline selection here.
- Race identity can be introduced here.
- Fragment interaction should be hinted, not fully opened.

## First five minutes

1. Player arrives at gate or broken road marker.
2. Roadwarden tells player to register.
3. Archivist identifies player as Remnant.
4. Race identity is acknowledged.
5. Player gets sent to inspect a road anomaly, not to kill ten wolves like every MMO apparently signed the same cursed contract.
