# Oraxen Resource Pack Test

## Commands

```text
/oraxen reload items
/oraxen reload pack
/oraxen pack send <player>
```

## Client checks

- Player receives pack prompt or automatic pack.
- Pack downloads successfully.
- No missing model/texture visuals.
- Prop icons display correctly in inventory.
- Furniture models display correctly when placed.

## Console checks

Look for:

```text
missing texture
missing model
failed to generate
failed to upload
unknown mechanic
invalid configuration
```
