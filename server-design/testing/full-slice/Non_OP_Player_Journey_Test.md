# Non-OP Player Journey Test

## Purpose

Prove that the starter slice works for a real player, not only for admins who have enough permissions to accidentally hide every problem.

## Test player setup

- OP: false
- Creative: false
- Admin permissions: none
- Previous quest state: cleared
- Previous FragmentEngine state: cleared or documented

## Journey checkpoints

| Checkpoint | Expected result | Pass/Fail | Evidence |
| --- | --- | --- | --- |
| Login | Player arrives safely |  |  |
| Lantern's Rest | Region protections work |  |  |
| Registrar | Dialogue starts |  |  |
| Registry state | FragmentEngine stores identity |  |  |
| Archivist | RE:FRAGMENT intro works |  |  |
| Bent Road | Route is readable |  |  |
| Anomaly site | Tutorial event triggers |  |  |
| Combat | Starter mob is survivable |  |  |
| Return | Quest can complete |  |  |
| Relog | State persists |  |  |

## Player feedback prompts

Ask the tester:

1. Did you know where to go next?
2. Did the story make sense without reading external lore?
3. Did anything feel broken or unfinished in a bad way?
4. Was combat too easy, fair, or too hard?
5. Did any UI/message feel confusing?
