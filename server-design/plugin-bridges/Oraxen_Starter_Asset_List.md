# Oraxen Starter Asset List - Lantern Marches

## Purpose

List visual/resource-pack assets needed for the starter vertical slice.

| --- | --- |
| registry_ledger | Registry Hall interaction prop/icon |
| road_marker_bent | The Bent Road marker |
| lantern_marches_token | Quest item / local mark visual |

### Phase 2 - Nice to have

| Asset | Use |
| --- | --- |
| cartotheca_map_case | Route scout / Cartotheca prop |

| ash_mile_marker | Ash Mile route marker |

| severed_black_pin | Suspicious traveler hint |

## Visual language

- Registry/Archivist assets: ivory, gold, ink, teal accent.
- Road assets: worn stone, iron, old paint, weak lantern glow.
- Memory/anomaly assets: cyan/teal glass, amethyst, subtle distortion.
- Severed hint assets: dark metal, broken circle, purple-black accent.

## Important boundary

Oraxen defines visuals and resource-pack objects. It does not define player identity, Fragments, Intent, Disciplines, or progression.
