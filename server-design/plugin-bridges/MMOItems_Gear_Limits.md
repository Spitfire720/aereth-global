# MMOItems Gear Limits - Starter Slice

## Purpose

Define what MMOItems/MythicLib can safely handle in the Lantern Marches.

## Allowed use

- Basic starter weapons.
- Basic starter tools.
- Low-tier armor.
- Quest reward items if they are physical objects.
- Minor stat variations for gear.

## Not allowed

- Fragments as item types.
- Intent as item stats.
- Disciplines as item classes.
- Titles as item cosmetics.
- Race identity as gear.
- Permanent character state stored only through gear.

## Starter gear examples

| Item | Type | Role |
| --- | --- | --- |
| Roadwarden's Shortblade | Weapon | Simple melee starter weapon |
| Ledgerbound Knife | Weapon | Light weapon with lore flavor |
| Marcher's Wraps | Armor | Basic starter armor |
| Wickpost Buckler | Shield/offhand | Optional defensive item |
| Bent Road Charm | Trinket/quest item | Should not grant Fragment power |

## Reward tuning

Starter gear should be functional, not defining.

The player's identity should come from race, Remnant condition, future Discipline, Title, Fragment interactions, and Intent, not from a sword that claims to understand ontology. Because obviously the sword would lie.
