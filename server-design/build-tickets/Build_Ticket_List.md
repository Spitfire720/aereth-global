# Build Ticket List - Lantern Marches Starter Slice

## A. World build tickets

### LM-BUILD-001 - Lantern's Rest blockout

Create the first hub blockout with gate, registry, market, lantern tower, and road exits.

Acceptance criteria:

- Spawn/arrival point exists.
- Registry Hall is visible from arrival route.
- North and southwest roads are readable.
- Hub feels safe but visibly damaged.

### LM-BUILD-002 - The Bent Road

Create the first route leaving Lantern's Rest.

Acceptance criteria:

- Three route markers exist.
- One repeated/bent marker is visually obvious.
- Small combat spaces exist off-road.

### LM-BUILD-003 - Mirrorfen edge

Create the reflection anomaly wetland.

Acceptance criteria:

- Half-submerged road stones.
- Reflection landmark or visual trick.
- Space for wisp mobs.

### LM-BUILD-004 - Old Wick Post

Create abandoned outpost combat pocket.

Acceptance criteria:

- Ruined watch post.
- Dead lanterns.
- Satchel/recovery interaction point.
- 2-3 mob spawn positions.

### LM-BUILD-005 - Hollowglass Pool

Create controlled anomaly arena.

Acceptance criteria:

- Clearly distinct visual from normal terrain.
- Small safe edge for event dialogue.
- Spawn positions for Shardlings.
- Return route is clear.
