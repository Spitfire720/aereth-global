# Build 01 Release Gate

## Build name

Lantern Marches Build 01 - Starter Vertical Slice

## Purpose

Build 01 is the first playable server slice. It proves that the Aereth server can combine canon, build design, plugin configuration, RPG state, NPC flow, starter combat, and visual props into one coherent player experience.

## Scope

Included:

- Lantern's Rest blockout
- The Bent Road route
- Hollowglass Pool or equivalent low-risk anomaly site
- WorldGuard protections
- Starter NPCs and dialogue
- First questline
- Starter mobs
- Starter Oraxen props
- FragmentEngine registration/tutorial bridge
- Full non-OP test

Not included:

- Full continent build
- Full economy
- Full Disciplines
- Full profession loop
- Advanced combat balancing
- Final art pass
- Public website/wiki release

## Required pass conditions

1. A new non-OP player can complete the starter slice without admin help.
2. FragmentEngine owns permanent RPG state.
3. BetonQuest owns only dialogue/quest flow.
4. MythicMobs encounters are survivable and thematically correct.
5. Oraxen props load without blocking client play.
6. WorldGuard protects the starter space without blocking intended interactions.
7. The experience introduces Aereth, RE:FRAGMENT, Remnant condition, and the Lantern Marches without contradiction.

## Build 02 unlock condition

Build 02 may begin only after Build 01 is marked:

- Passed with known minor issues, or
- Passed and ready for Build 02

Critical issues must become fix tickets before Build 02.
