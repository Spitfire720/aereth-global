# Repo Archive

This folder is for repo-level archived material.

## What belongs here

- Superseded docs.
- Old generated plans.
- Historical handoffs.
- Dead drafts.
- Previous cleanup backups.

## What does not belong here

- jars
- server backups
- database files
- player data
- secrets
- resource pack builds
- model exports

Archive material should explain why it was archived.

A pile with a label is still a pile. Give it context.
