# Aereth Website

This folder is reserved for the public Aereth website and wiki.

## Direction

The website should feel like an Aereth interface, not a generic Minecraft server landing page.

It should support:

- public homepage
- join/play section
- world preview
- Disciplines page
- Fragments page
- Intent page
- wiki index
- updates/news

## Planned structure

```text
website/
  src/
  public/
  content/
    wiki/
    updates/
  README.md
```
