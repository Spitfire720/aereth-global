# Aereth Docs

This folder holds active project documentation that belongs in GitHub.

Notion is still the planning and canon brain. GitHub docs are for versioned, repo-adjacent truth.

## Active docs

- `REPO_STRUCTURE.md`
- `ARCHITECTURE.md`
- `OPERATIONS.md`
- `WEBSITE_AND_WIKI.md`
- `OBSIDIAN_CLEANUP_PLAN.md`

## What belongs here

- Architecture rules.
- Repository structure rules.
- Build/deployment operations.
- Website/wiki implementation direction.
- Obsidian cleanup plan.
- Developer handoff notes that should version with code.

## What does not belong here

- Large binaries.
- Jars.
- Generated packs.
- Server backups.
- Player data.
- Secrets.
- Raw lore dumps that belong in Obsidian/Notion.
