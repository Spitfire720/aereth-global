# Aereth Map Visual Direction

## Visual Identity

Aereth maps should feel like premium MMORPG atlas art:

- ornate dark-fantasy frame
- antique parchment mixed with cosmic night texture
- gold and ivory serif labels
- blue-teal magical route lines
- purple-black fracture energy
- celestial and arcane motifs
- fogbound outer world
- readable high-level geography

## Map Family

1. Macro World Map
2. Detailed World Atlas Map
3. Regional Progression Map
4. Political & Cultural Atlas Map

## Production Rule

Generated image labels are acceptable for concept work, but final public assets should eventually use manual label overlays.

Recommended production process:

1. Use generated map as background.
2. Remove or cover unstable generated microtext if needed.
3. Add final labels manually in Figma, Photoshop, Photopea, or similar.
4. Use a consistent icon set.
5. Keep map layers separated where possible:
   - background art
   - labels
   - routes
   - faction overlays
   - legend
   - website crop versions

## Website Crops Needed Later

- Home hero crop
- World page full-width crop
- Wiki thumbnail crop
- Mobile-safe crop
- Continent page detail crops
- Faction influence detail crops

## Typography Direction

Use elegant fantasy serif for titles and region labels. Avoid overloading the map with tiny unreadable text. Aereth should feel ancient, not like someone printed a spreadsheet on parchment because suffering needed columns.
