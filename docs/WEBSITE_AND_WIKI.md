# Website and Wiki

## Purpose

The `website/` folder should become the public Aereth website and wiki app.

The site should feel like an Aereth interface, not a generic Minecraft server website.

## Main website sections

- Home
- Play / Join
- World
- Disciplines
- Fragments
- Intent
- Map
- Wiki
- Updates

## MVP website scope

- Homepage
- Play / join section
- World preview
- Disciplines page
- Fragments page
- Intent page
- Wiki index
- Updates placeholder

## MVP wiki scope

- What is Aereth?
- How to Join
- First 30 Minutes
- Races
- Progression
- Disciplines
- Fragments
- Intent
- Commands
- Glossary

## Expected content structure

```text
website/
  src/
  public/
  content/
    wiki/
    updates/
  README.md
```

## Copy rules

- Short lines.
- Strong atmosphere.
- Specific, not generic.
- No fake-deep paragraphs.
- No stock MMO copy.
