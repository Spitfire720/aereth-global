# Aereth Roadmap

## Roadmap principle

Build base systems before quests.

No quest/content expansion until the RPG backend is stable enough to carry it.

## Phase 0: Project control and cleanup

Status: active.

- Clean Notion into v2 structure.
- Clean GitHub repo root.
- Align `website/` with public site/wiki plan.
- Keep Drive as storage, not project brain.
- Archive stale material without deleting history.

Status: mostly passed.
