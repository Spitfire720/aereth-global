# Build 01 - Lantern Marches Implementation

Status: Draft implementation layer  
Related region: [[The Lantern Marches]]  
Hub: Lantern's Rest  
Purpose: first playable server slice  

## What this build proves

Build 01 proves that Aereth can turn canon into playable server structure.

It connects:

- lore onboarding
- safe starter hub
- WorldGuard protection
- BetonQuest dialogue
- MythicMobs starter enemies
- Oraxen visual props
- FragmentEngine ownership boundaries

## Playable loop

1. Player arrives in Lantern's Rest.
2. Player speaks to Archivist Maerin Vale at The Quiet Ledger.
3. Player is registered as a Remnant.
4. Player is sent to The Bent Road.
5. Player clears Roadstrays.
6. Player investigates Hollowglass Pool.
7. Player encounters Hollowglass Wisp.
8. Player returns to The Quiet Ledger.
9. Player receives the next progression hook.

## Canon interpretation

The player is not given full truth.

They learn:

- Aereth exists after RE:FRAGMENT.
- Remnant is an existential condition, not a race.
- Records matter because memory and reality are damaged.
- The Lantern Marches are safer than the wider world, not truly safe.
- Fragments and anomalies change perception and responsibility before raw mechanics.

## Plugin ownership

- FragmentEngine owns RPG state.
- BetonQuest coordinates story flow.
- MythicMobs owns enemy behavior.
- Oraxen owns visual props.
- WorldGuard owns safety/protection.
- PlaceholderAPI displays state only.

## Build 01 exclusions

Not included yet:

- full Fragment tutorial
- Discipline unlocks
- profession systems
- economy loop
- full race questlines
- full Lantern Marches map
- world bosses
- mutated Disciplines

This is deliberate. Scope creep is how servers become abandoned Discord announcements.
