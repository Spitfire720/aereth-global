# Starter Vertical Slice - Lantern Marches

## Status

Server implementation bridge, v1.

## Canon source

The Lantern Marches are the starter subregion of Aereth. They sit between Valterra and Lyssara influence, with Lantern's Rest as the main starter hub.

## Server implementation target

The first vertical slice should make the following playable:

- Arrival at Lantern's Rest.
- Registration as a Remnant.
- Race identity acknowledgement.
- First old-road route into The Bent Road.
- First memory anomaly in Mirrorfen.
- First combat pocket at Old Wick Post.
- First controlled Fragment disturbance at Hollowglass Pool.
- Return to Lantern's Rest with local recognition.

## Plugin responsibilities

| Plugin/system | Role in starter slice |
| --- | --- |
| FragmentEngine | RPG state, Fragment/Intent/title-ready identity state |
| BetonQuest | Dialogue and starter questline |
| MythicMobs | Starter mobs and controlled event wave |
| Oraxen | Visual props and icons |
| WorldGuard | Region protection and safe/unsafe boundaries |
| PlaceholderAPI | Display bridge for player state |
| MMOItems/MythicLib | Physical starter gear only |

## Key canon boundary

Fragments are not loot, items, or raw power increases. They change perception, access, responsibility, consequence, pressure, and world interaction.

## First player experience

The first 30-60 minutes should teach:

1. Aereth is wounded but alive.
2. RE:FRAGMENT happened.
3. The player is a Remnant by condition.
4. Race is separate from Remnant state.
5. Fragments are serious and not simple upgrades.
6. The West is safer, but not safe.
7. The world is much larger than Lantern's Rest.
