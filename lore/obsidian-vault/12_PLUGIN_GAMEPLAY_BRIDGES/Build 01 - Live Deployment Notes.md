# Build 01 - Live Deployment Notes

Status: Draft deployment layer.

Region: [[The Lantern Marches]]
Hub: [[Lantern's Rest]]
Purpose: first playable starter vertical slice.

## Deployment principle

The server implementation must prove a small loop before expanding the region.

The first live slice is:

1. Lantern's Rest.
2. Registration / archive post.
3. The Bent Road.
4. One low-risk combat encounter.
5. Hollowglass Pool anomaly.
6. Return to Lantern's Rest.

## Plugin ownership

- WorldGuard owns region protection.
- BetonQuest owns dialogue and quest flow.
- MythicMobs owns enemies and encounter behavior.
- Oraxen owns visual props and resource-pack presentation.
- FragmentEngine owns RPG state and player identity.
- PlaceholderAPI only displays state.

## Canon safety

Fragments remain existential/perceptual/reality-pressure states, not items.

The first starter anomaly must show Fragment disturbance without making Fragments feel like loot.

## Build 01 acceptance criteria

- Player understands where they are.
- Player understands Aereth is post-RE:FRAGMENT.
- Player can follow the first route.
- Player encounters low-risk distortion.
- Player returns to safety.
- Console remains clean enough to continue development.
