# 🎨 Visual Direction Overview

Aereth visual identity combines dark fantasy atlas, memory fracture, neon blue/cyan UI accents, gold titles, red/pink danger and plus symbols, ancient cartography, cosmic void textures, and Minecraft-readable silhouettes.

## UI color canon
- Titles: blue neon
- Numbers: red
- Plus signs: pink
- XP / Pressure / Erasure: white
- Health title: blue

## Map style
Premium fantasy MMO map, ornate border, dark parchment, cosmic blue fracture lines, gold labels, fogbound continents.
